# Проект: Место

**Общее**

Проект "Mesto" разработан в рамках обучения в Яндекс Практикуме по программе Вэб-разработка. Проект является точной копией [макета](https://www.figma.com/file/bjyvbKKJN2naO0ucURl2Z0/JavaScript.-Sprint-5?node-id=0%3A1&t=YV6jblAXqrXoc6kc-0). В процессе верстки макета использовались знания, полученные после прохождения 5ого спринта программы Вэб-разработка, а именно:

* Работа с DOM;
* Разработка визуального интерфейса для разных устройств (адаптивная верстка);
* Открытие/закрытие попапов ("Редактирование профиля", "Добавление новой карточки", "Превью карточки");
* Удаление элемента
* Event для лайка карточки
* Управление содержимым с помощью JS;
